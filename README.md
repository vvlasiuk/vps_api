# VPS API Confirmation Server

## Опис
FastAPI сервер для підтвердження команд через HTML‑кнопки, з інтеграцією RabbitMQ та MariaDB, дворівневою системою токенів і логуванням помилок.

### Основні можливості
- Прийом команд через POST /command (тимчасові токени)
- Відправка команд у RabbitMQ (input.queue)
- Видача тимчасових токенів через POST /token (по master-токену)
- Управління контекстом через /context
- Логування помилок у файл та sys_error.queue
- Адміністрування master-токенів через CLI

### Старт
1. Швидкий старт (Windows):
   Запустіть скрипт
   ```bat
   setup.bat
   ```
   Він створить віртуальне середовище (venv) та встановить залежності автоматично.

2. Встановіть залежності вручну (альтернатива):
   ```bash
   pip install -r requirements.txt
   ```
3. Налаштуйте змінні середовища для MariaDB та RabbitMQ 
4. Проведіть міграції (alembic або вручну):
   Увага!
   Після налаштування змінних середовища і перед запуском сервера обов’язково виконайте команди з розділу міграцій (Alembic) цього README, щоб створити всі необхідні таблиці у базі даних.

    - Якщо використовується Alembic, виконайте команду:
       ```bash
       alembic upgrade head
       ```
       Це створить необхідні таблиці у вашій базі даних згідно з моделями.
    - Якщо alembic не використовується, створіть таблиці вручну відповідно до моделей у app/models.py (SQL-структура описана у класах Base, MasterToken, Token, Context).
5. Запустіть сервер:
   ```bash
   uvicorn app.main:app --host 0.0.0.0 --port 8000 --ssl-keyfile=key.pem --ssl-certfile=cert.pem
   ```
6. Для керування master-токенами використовуйте CLI:
   ```bash
   python manage_tokens.py --help
   ```

### Структура RabbitMQ
- **input.queue**: приймає команди
- **sys_error.queue**: приймає помилки

### Безпека
- Тільки HTTPS (самопідписані сертифікати)
- Авторизація через токени
